## Decisions Made
- Used TipTap module augmentation (`declare module '@tiptap/core'`) for typed custom commands -- avoids unsafe casts and gives autocomplete for setDiffDecorations/clearDiffDecorations
- Implemented textOffsetToPos helper that walks ProseMirror doc nodes for position mapping -- more robust than indexOf on textContent for complex documents
- Diff decorations only target 'added' hunks since 'removed' text has no position in the current document; removed line count shown in DiffOverlay summary instead
